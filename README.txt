JSS Scrolling is an addon to the JS-SS Plugin, working with the animation side of CSS through Javascript.

== Description ==

JSS Scrolling is the transforming StyleSheet end of JS-SS, it makes adding website animations easy but also creates your animation to the pixel this is a nicer way of creating an animation. Instead of performing a full fast animation when you sroll to a certain part of the page it animates as you scroll and when you stop scrolling it stops animating. It works both in forward motion and reverse as you scroll up or down a page.

== Installation ==

1. Install the plugin through Wordpress Plugin library
1. Activate the plugin through the 'Plugins' menu in WordPress
